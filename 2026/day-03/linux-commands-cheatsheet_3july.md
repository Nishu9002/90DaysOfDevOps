Process management commands:

bg - this resumes a stopped job in the background (bg %1)
fg - this brings a background job to the foreground
kill - it force stops a process
top - it shows the active running process with their process ids, cpu usage and memory usage

Networking troubleshooting commands:

curl - it is used to transfer data to or from a server using various protocols like https or https. It is basically used for testing APIs, downloading files, interacting with web services from the terminal.

ifconfig - used to display and config network interfaces. its replaced by (ip addr) to show the ip address of the network

netstat - to monitor the network connections, it shows the user all the applications using the network, tcp/udp connections, listening ports, etc.

nslookup - to diagnose dns server issues, to check the domain names, ip addresses and resolve dns queries

ping - to check the reachability of a host or an IP address on a network. 

ssh - it is used to connect securely to remote systems over the network.
