What is your understanding of DevOps and Cloud Engineering?
    Devops engineering is a practice that helps bridge the gap between the developers and the consumers. It ensures that the application is up on the server, manages for traffic and also maintains scalability.
    Basically, in a software development lifecycle, how code moves from development to testing to deployment, it is all automated by devops engineers.
    And cloud engineering is about managing cloud services like servers, storage and network for running the applications.
